Chatbot: Реализуем слой данных чат-бота на базе БД Microsoft SQL Server (завершение)

В конце урока мы дописываем и запускаем Telegram-бота, в котором слой доступа данных для хранения задач чат-бота начинает использовать настоящую базу данных MS SQL Server вместо словаря в памяти.
Плюсом данного изменения является повышение отказоустойчивости системы: если раньше при перезапуске сервиса Web API, например при перезагрузке компьютера, все данные о запланированных напоминаниях пропадали, в новой имплементации данные остаются в безопасности, более того, появляется возможность настроить периодическое резервное копирование (Fault-tolerant Design).
