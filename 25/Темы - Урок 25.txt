Chatbot: Создание интерфейса доступа к слою данных через Web API и написание клиентской библиотеки (.NET Standard) для работы с Web API (завершение)

В конце урока мы дописываем и запускаем Telegram-бота, в котором слой доступа данных вынесен в отдельный сервис Web API (доступный по HTTP-протоколу).
Плюсом данной организации является обеспечение слабой связности модуля доступа к данным и самого приложения (Low Coupling Principle)
По факту, у нас появляется два приложения - ASP.NET Core Web API сервис для хранилища задач чат-бота и собственно консольное приложение бота, которое обращается к хранилищу посредством HTTP-протокола.
